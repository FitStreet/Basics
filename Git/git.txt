локальный репозиторий - папка .git на внутри нашего ноута
удаленный репозиторий - папка на сервере(которой мы создаем в GitHub)

git init - создает git репозиторий из текущей папки
git add . - добавляет файлы и папки для оттслеживания(всегда пишем перед commit)
git commit -m "тут пишем сообшение" - создает новую версию (сохраняет изменения), если были изменения

git status - показывает состояние файлов
git branch - показывает все сетки
